# apps/

O diretório `apps/` contém as aplicações cliente nativas do projeto Kairos. Cada subdiretório representa uma plataforma-alvo distinta ou um kit compartilhado entre plataformas, complementando a interface web e a CLI do projeto.

## Subdiretórios

| Diretório   | Descrição |
|-------------|-----------|
| `android/`  | App Android nativo integrado via Capacitor (Kotlin/Java). |
| `ios/`      | App iOS nativo desenvolvido em Swift com suporte a Xcode. |
| `shared/`   | Kit compartilhado entre plataformas nativas — OpenClawKit em Swift. |
