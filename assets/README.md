# assets/

O diretório `assets/` contém os assets visuais do projeto Kairos, usados em instaladores, documentação e na interface.

## Conteúdo

| Item | Descrição |
|---|---|
| `logo.svg` | Logo principal do Kairos em formato vetorial |
| `avatar-placeholder.svg` | Placeholder de avatar para a interface |
| `dmg-background.png` | Imagem de fundo do instalador DMG (macOS) — alta resolução |
| `dmg-background-small.png` | Versão reduzida da imagem de fundo do DMG |
| `metadata.json` | Metadados dos assets (nomes, dimensões, etc.) |
| `chrome-extension/` | Assets específicos para a extensão Chrome |
