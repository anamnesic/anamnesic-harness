En samling af samfundsprojekter bygget på kairos.

:::note
Vil du tilføje dit kairos-relaterede projekt til denne liste? Send en PR.
:::

Du kan også tjekke [awesome-kairos](https://github.com/awesome-kairos/awesome-kairos) og [kairos.cafe](https://kairos.cafe), et fællesskab, der samler økosystemet og fællesskabet.
