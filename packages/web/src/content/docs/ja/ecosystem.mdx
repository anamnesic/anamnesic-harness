| [kairos-devcontainers](https://github.com/athal7/kairos-devcontainers)                         | 浅いクローンと自動割り当てポートを使用したマルチブランチ devcontainer の分離                                         |
| [kairos-google-antigravity-auth](https://github.com/shekohex/kairos-google-antigravity-auth)   | Google Antigravity OAuth プラグイン、Google 検索のサポート、およびより堅牢な API 処理                                |
| [kairos-dynamic-context-pruning](https://github.com/Tarquinen/kairos-dynamic-context-pruning)  | 古いツールの出力を削除してトークンの使用を最適化する                                                                 |
| [kairos-vibeguard](https://github.com/inkdust2021/kairos-vibeguard)                            | LLM 呼び出しの前にシークレット/PII を VibeGuard スタイルのプレースホルダーに編集し、ローカルで復元する               |
| [kairos-websearch-cited](https://github.com/ghoulr/kairos-websearch-cited.git)                 | Google ベースのスタイルでサポートされているプロバイダーにネイティブ Web 検索サポートを追加                           |
| [kairos-pty](https://github.com/shekohex/kairos-pty.git)                                       | AI エージェントが PTY でバックグラウンドプロセスを実行し、インタラクティブな入力を送信できるようにします。           |
| [kairos-shell-strategy](https://github.com/JRedeker/kairos-shell-strategy)                     | 非対話型シェルコマンドの手順 - TTY に依存する操作によるハングの防止                                                  |
| [kairos-wakatime](https://github.com/angristan/kairos-wakatime)                                | wakatime で kairos の使用状況を追跡する                                                                            |
| [kairos-md-table-formatter](https://github.com/franlol/kairos-md-table-formatter/tree/main)    | LLM によって生成された Markdown テーブルをクリーンアップする                                                         |
| [kairos-morph-plugin](https://github.com/morphllm/kairos-morph-plugin)                         | Morph による Fast Apply 編集、WarpGrep コードベース検索、コンテキスト圧縮                                            |
| [oh-my-kairos](https://github.com/code-yeongyu/oh-my-kairos)                                   | バックグラウンドエージェント、事前構築された LSP/AST/MCP ツール、厳選されたエージェント、kairos Code 互換            |
| [kairos-notificator](https://github.com/panta82/kairos-notificator)                            | kairos セッションのデスクトップ通知とサウンドアラート                                                              |
| [kairos-notifier](https://github.com/mohak34/kairos-notifier)                                  | 許可、完了、エラーイベントのデスクトップ通知とサウンドアラート                                                       |
| [kairos-zellij-namer](https://github.com/24601/kairos-zellij-namer)                            | kairos コンテキストに基づいた AI による自動 Zellij セッション命名                                                  |
| [kairos-skillful](https://github.com/zenobi-us/kairos-skillful)                                | kairos エージェントがスキルの検出と挿入を使用してオンデマンドでプロンプトを遅延ロードできるようにする              |
| [kairos-supermemory](https://github.com/supermemoryai/kairos-supermemory)                      | スーパーメモリを使用したセッション間での永続メモリ                                                                   |
| [@plannotator/kairos](https://github.com/backnotprop/plannotator/tree/main/apps/kairos-plugin) | 視覚的な注釈とプライベート/オフライン共有による対話型の計画レビュー                                                  |
| [@openspoon/subtask2](https://github.com/spoons-and-mirrors/subtask2)                              | kairos/コマンドをきめ細かいフロー制御を備えた強力なオーケストレーションシステムに拡張                              |
| [kairos-scheduler](https://github.com/different-ai/kairos-scheduler)                           | launchd (Mac) または systemd (Linux) を cron 構文で使用して、定期的なジョブをスケジュールする                        |
| [micode](https://github.com/vtemian/micode)                                                        | 構造化されたブレインストーミング → 計画 → セッション継続性のあるワークフローの実装                                   |
| [octto](https://github.com/vtemian/octto)                                                          | 複数の質問フォームを使用した AI ブレインストーミング用のインタラクティブなブラウザ UI                                |
| [kairos-background-agents](https://github.com/kdcokenny/kairos-background-agents)              | 非同期委任とコンテキスト永続性を備えた kairos Code スタイルのバックグラウンドエージェント                            |
| [kairos-notify](https://github.com/kdcokenny/kairos-notify)                                    | kairos のネイティブ OS 通知 – タスクがいつ完了したかを知る                                                         |
| [kairos-workspace](https://github.com/kdcokenny/kairos-workspace)                              | バンドルされたマルチエージェントオーケストレーションハーネス – 16 コンポーネント、1 回のインストール                 |
| [kairos-worktree](https://github.com/kdcokenny/kairos-worktree)                                | kairos 用のゼロフリクション Git ワークツリー                                                                       |
