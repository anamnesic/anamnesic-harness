Una colección de proyectos comunitarios construidos en kairos.

:::note
¿Quieres agregar tu proyecto relacionado con kairos a esta lista? Envíe un PR.
:::

También puedes consultar [awesome-kairos](https://github.com/awesome-kairos/awesome-kairos) y [kairos.cafe](https://kairos.cafe), una comunidad que agrega el ecosistema y la comunidad.
