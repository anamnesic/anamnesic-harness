# .kairos

Configuracao local do Kairos, comandos, skills, temas, ferramentas e glossarios internos.

## Conteudo
- Subpastas: `agent/`, `command/`, `glossary/`, `plugins/`, `skills/`, `themes/`, `tool/`.
- Arquivos principais: `kairos.jsonc`, `tui.json`, `env.d.ts`.

## Quando mexer aqui
Atualize quando adicionar comandos, skills, temas, ferramentas ou configuracoes locais do Kairos.
