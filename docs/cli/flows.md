# `kairos tasks flow`

Flow commands are subcommands of `kairos tasks`, not a standalone `flows` command.

```bash
kairos tasks flow list [--json]
kairos tasks flow show <lookup>
kairos tasks flow cancel <lookup>
```

For full documentation see [Task Flow](/automation/taskflow) and the [tasks CLI reference](/cli/tasks).
