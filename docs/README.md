# docs/

Diretório de documentação principal do Kairos. Contém documentos de arquitetura, guias operacionais, referências técnicas, relatórios de análise e subdiretórios de documentação especializada por domínio. É o ponto de partida para entender o projeto, suas decisões de design e seu estado atual.
